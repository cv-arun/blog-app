# blog-app

## Run Locally

Clone the project

```bash
  git clone https://github.com/cv-arun/blog-app.git
```

Go to the project directory backend

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Go to the project directory frontend

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
